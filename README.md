# Daily Verse Reading

iOS app for daily Bible reading with streak tracking, favorites, and shareable verse cards.

**Repository:** https://github.com/Luknarz/DailyVerse — **this is the single source of truth.** Always pull before working and push when done so everyone has the latest code.

## For Collaborators

- **Pull before you start:** `git pull origin main`
- **Push when you're done:** `git push origin main`
- See [CONTRIBUTING.md](CONTRIBUTING.md) for the full workflow.

## Project Structure

- `DailyVerse/` — Main app (Swift/SwiftUI)
- `DailyVerseTests/` — Unit tests
- `scripts/` — App Store submission automation
- `DailyVerse_Screenshots/` — App Store screenshots

## App Store Submission

Documentation and scripts for submitting to the App Store are in the repo. Start with [QUICK_START_SUBMISSION.md](QUICK_START_SUBMISSION.md).

## Setup

1. Clone: `git clone https://github.com/Luknarz/DailyVerse.git`
2. Open `DailyVerse.xcodeproj` in Xcode
3. Select the DailyVerse scheme and run
